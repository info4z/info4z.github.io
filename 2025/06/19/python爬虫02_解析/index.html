JSONPath和XPath语法对比

xpath jsonpath 描述
/ $ 根元素
/ . 子元素
// .. 递归下降，从E4X借鉴
* * 通配符，表示所有元素
[] [] 子元素操作符
. @ 当前元素
n/a () 脚本表达式，使用在脚本引擎下面
n/a [start:end:step] 数组分割操作，从ES4借鉴
[] ?() 应用过滤表达式
.. n/a 父元素
@ n/a 属性访问字符
` ` [,]
() n/a Xpath分组

举例对比

xpath jsonpath 描述
/store/book/author $.store.book[*].author 商店所有书的作者
//author $..author 所有的作者
/store/* $.store.* store下的所有元素
/store//price $.store..price store中所有东西的price
//book[3] $..book[2] 第三本书
//book[last()] $..book[(@.length-1)] 最后一本书
//book[position()<3] $..book[0,1] 或 $..book[:2] 前两本书
//book[isbn] $..book[?(@.isbn)] 过滤所有包含isbn的书
//book[price<10] $..book[?(@.price<10)] 过滤出价格低于10的书
//* $..* 所有元素

三、BeautifulSoup

BeautifulSoup也被称为bs4，和lxml一样，是一个html的解析器，主要功能也是解析和提取数据。没有lxml效率高，但是胜在接口设计更人性化，使用方便。

（一）安装使用

需要安装bs4的库

基本使用

（二）基本语法

根据标签名获取信息

属性 描述
soup.标签名 获取第一个指定类型的标签，tag
soup.标签名.name 获取第一个指定类型的标签的标签名，感觉没啥用
soup.标签名.attrs 获取第一个指定类型的标签的所有属性名和属性值，dict

通过函数获取标签：find()，find_all()，select()

函数 描述
soup.find('标签名') 获取第一个指定类型的标签，tag
soup.find('标签名', 属性名='属性值') 获取符合属性条件的指定类型的标签，tag
soup.find_all('标签名') 获取所有的指定类型的标签，list
soup.find_all(['标签名', '标签名2']) 获取所有的指定类型的标签，list
soup.find_all('标签名', limit=数字) 获取前几个指定类型的标签，list
soup.select('bs4选择器') 根据选择器获取标签，list

bs4选择器

选择器 描述
元素选择器 如：select(‘a’)，获取所有的a标签
如：select(‘div,p’)，获取所有div标签和p标签
类选择器 如：select(‘.c1’)，获取所有的class=”c1”的标签
id选择器 如：select(‘#a1’)，获取id为a1的标签
属性选择器 如：select(‘li[class]’)，获取全部有class属性的li标签
如：select(‘li[class=”c1”]’)，获取class为c1的li标签
层级选择器 如：select(‘div p’)，获取div下的所有p标签
如：select(‘div>p’)，获取div的子代p标签

获取标签的属性

属性/方法 描述
标签.name 获取标签的名字，没啥用
标签.attrs 获取标签的所有属性，dict
标签.attrs.get('属性名') 获取标签的属性值，list
标签.get('属性名') 获取标签的属性值，list
标签['属性名'] 获取标签的属性值，list
标签.string 获取标签的内容，只能是直接内容
标签.get_text() 获取标签的内容，直接内容或者嵌套内容都可以
